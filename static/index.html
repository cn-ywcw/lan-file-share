❓ 帮助与使用说明

📖 快速入门

打开浏览器访问本服务地址，即可在同个局域网内任意设备间共享文件。

  • 本机访问：http://localhost:8080
  • 局域网访问：http://192.168.x.x:8080
  • 手机、平板、其他电脑均可通过浏览器直接访问，无需安装任何软件。

📤 上传文件

  • 拖拽上传：将文件或文件夹从电脑拖入左侧上传区域
  • 选择文件：点击"选择文件"按钮，可多选
  • 上传文件夹：点击"选择文件夹"按钮，保留目录结构
  • 上传时显示进度条，完成后自动刷新文件列表

📂 管理文件

  • 浏览：双击文件夹进入，双击文件预览
  • 下载：列表视图下点击 ⬇️ 按钮，右键菜单也可下载；文件夹会自动打包为 ZIP
  • 预览：支持图片、视频、音频、PDF、文本/代码、ZIP 压缩包内容
  • 删除：右键菜单或列表中的 🗑️ 按钮
  • 新建文件夹：点击工具栏 📁+ 新建
  • 搜索：在搜索框输入文件名实时过滤
  • 视图切换：网格视图 ▦ / 列表视图 ☰

🖼️ 图片画廊

  • 打开一张图片后，可使用 ◀ ▶ 按钮切换当前目录中的上一张/下一张图片
  • 也支持键盘 ← → 方向键切换
  • 支持的格式：JPG、PNG、GIF、SVG、WebP、BMP

📦 压缩包预览

  • 双击 .zip 文件可查看其内部文件清单
  • 支持树形目录结构展示
  • RAR / 7z / TAR 格式暂不支持在线浏览，请下载后使用本机软件解压

🔗 分享链接

  • 右键文件 →「🔗 Share」或在预览窗口点击 🔗 按钮
  • 可选设置访问密码和过期时间
  • 生成后会在左侧边栏列出所有活跃分享，可随时删除
  • 分享链接格式：http://IP:端口/share/<token>

⌨️ 快捷键

  • ← → — 图片预览时切换上一张/下一张
  • Esc — 关闭当前弹窗
  • Ctrl+R — 刷新文件列表

🔒 安全说明

  • 默认情况下服务没有密码保护，同局域网内所有设备均可访问
  • 如需保护，启动时添加 --user admin --pass 你的密码
  • 分享链接的密码经过哈希处理，传输采用 Basic Auth

💡 提示

  • 将所有文件放在共享目录（默认为 ./shared/）即可被浏览
  • 可指定其他目录：--dir D:\共享文件夹
  • 最大上传大小默认为 500MB，可通过 --max-upload 1000 修改
